# Site plugins (ITM)

Per-site bundles for the `it-infra-monitoring` product.

| Plugin | Purpose |
|--------|---------|
| `itm-site-inventory` | Devices, SNMP placeholders, servers/ISP/VoIP tree |
| `itm-site-topology` | DCN mimic, dashboard patch, SVG topology |
| `itm-site-integrations` | AD/LDAP, CMS/OPC DA metadata, roles |

Import via `POST /api/v1/platform/packages/import?packageId=<packageId>`.

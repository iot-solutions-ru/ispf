# Оператор PWA — контрольный список дыма для Android Chrome (BL-90)

Ручное принятие установленного режима оператора на Android. **Автоматическое освещение (CI):** Playwright Pixel 5 + предварительный просмотр (`e2e/pwa-android.spec.ts`) — манифест, ПО, ротация, автономный баннер с перезагрузкой ПО. Мобильная область просмотра + автономный кеш в `e2e/live-operator.spec.ts`.

## Предварительные условия

- HTTPS origin (prod `${ISPF_BASE_URL:-https://ispf.example.invalid}` or local tunnel with valid cert)
- Demo operator app available (`/?mode=operator&app=demo`)
- Устройство Android с Chrome 120+ (необязательно — только для подписания выпуска)

## Установить

1. Откройте сайт в Chrome → меню → **Установить приложение** / **Добавить на главный экран**.
2. Подтвердите, что имя приложения **ISPF** и его значок появятся на панели запуска.
3. Запуск с главного экрана — строка URL должна быть минимальной (автономное отображение).

## Оператор дымит (5 мин)

| Шаг | Ожидается | CI автоматизированный |
|------|----------|--------------|
| Приложение открывается по URL-адресу запуска оператора (`start_url` содержит `mode=operator`) | Загрузка оболочки, без дерева администрирования | ✓ манифест + оболочка оператора e2e |
| Войти (если требуется) | Сеанс сохраняется после перезапуска приложения | mock сессия в e2e |
| Навигация по вкладкам манифеста/панелям мониторинга | Верхняя панель + навигация видимы, безопасная область не обрезана (выемка) | ✓ Безопасная зона Pixel 5 + навигация |
| Поворот портрета ↔ пейзажа | Перекомпоновка макета; нет горизонтального overflow | ✓ тест вращения области просмотра |
| Включить режим полета | Офлайн-баннер + кэшированная оболочка (BL-91) | ✓ Автономная перезагрузка ПО (предварительный просмотр) |
| Отключить режим полета | Баннер очищается; данные обновляются без полной перезагрузки | ✓ тест повторного подключения |

## Примечания к регрессии

- Сервисный рабочий кэширует только оболочку; API использует сеть (в автономном режиме используется кеш localStorage)
- If install prompt missing: check `manifest.webmanifest`, `display: standalone`, icons 192/512
- Запишите модель устройства, версию Chrome и снимок экрана автономного баннера для примечаний к выпуску.

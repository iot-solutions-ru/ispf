### Идентификаторы

| Префикс | Значение | Пример |
| ------- | -------- | ------ |
| **Фаза N** | Тематическая дорожная карта волн (история + REQ-EX) | Фаза 20, Фаза 23 |
| **Спринт СН** | Единица доставки (~2 недели), единая нумерация S01… | С01, С18 |
| **ХФНН** | Исправление вне спринта | HF01 |
| **БЛ-НН** | Аудит кода задачи/совершенство | БЛ-80 |
| **PF-NN / FW-NN** | Требование прикладной платформы/фреймворка | ПФ-03 |

### Статусы (везде одинаково)

| Статус | Смысл |
| ------ | ----- |
| Готово | Приемка завершена |
| Частичный | Основа есть, хвост открыт |
| В процессе | Активная работа |
| Планируется | Не начато |
| Отменено | Отменено / заменено |
| Операции | Runbook/документы готовы; внедрение по запросу |

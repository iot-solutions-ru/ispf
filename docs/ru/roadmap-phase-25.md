## Определение готовности — общая оценка 10/10

Товар оценивается на **10/10**, при **одновременно**:

1. **20 ПРОИЗВОДСТВЕННЫХ драйверов** с интерактивным CI и 3 полевых пилотов — **готовые к работе пилоты** (BL-140 ✅); ожидается задержка клиента
2. **Пакет MES** — OEE + заказы + качество без индивидуального кода (BL-164…170)
3. **Агент ИИ** — ≥95 % сценариев регрессии, зелёный **с действующим LLM**, развертывание без правок (BL-177, BL-178) — **не выполнено** (CI: только схема; ночные незавершенные результаты)
4. **Историк** — 3-х уровневый, 1Б проверенных образцов под ключ (БЛ-159…162) под ключ
5. **Безопасность** — MFA + список управления доступом для каждой переменной + опция strict tenant isolation (BL-153…155)
6. **HMI** — mimic на 500 эл при 60 кадрах в секунду, автономное PWA 8 часов (BL-151, BL-152)
7. **Маркетплейс** — более 10 подписанных пакетов, 3 внешних партнера (BL-183, BL-184)
8. **Оценочная карта** — все 14 измерений ≥9,5, ни одного ≤8 (BL-189) — **не выполнено** (проверено по коду ~7,4/10; см. [COMPETITIVE_Scorecard.md](competitive-scorecard.md))
